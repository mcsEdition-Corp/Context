## English

# Privacy Policy for Context - Smart Tab Manager

*Last updated: January 2025*

## Overview
Context is a browser extension that helps you organize your tabs intelligently using AI. We are committed to protecting your privacy and being transparent about our data practices. **All your data stays on your device - we never store or access your information.**

## Information We Collect

### Tab Information
- **What we collect**: URLs, titles, and favicons of your open tabs
- **Why**: To analyze and organize your tabs into meaningful workspaces
- **Storage**: **Stored locally on your device only** using Chrome's secure storage API
- **Access**: Only you have access to this data

### API Key (Optional)
- **What we collect**: Your OpenAI API key (if you choose to provide one)
- **Why**: To enable AI-powered tab organization features
- **Storage**: **Encrypted and stored locally** in Chrome's secure storage on your device
- **Important**: Your API key is **never transmitted to us** - it goes directly from your browser to OpenAI's servers
- **Cost**: When you use AI features, you pay OpenAI directly. [View OpenAI GPT-4 Turbo pricing](https://openai.com/pricing)

### User Preferences
- **What we collect**: Your settings (auto-organize preferences, notification preferences, theme, etc.)
- **Why**: To customize your experience and remember your preferences
- **Storage**: **Stored locally on your device only**

## How We Use Information
1. **Tab Organization**: Analyze tab content locally to create intelligent workspaces
2. **Persistence**: Save workspace configurations locally for future sessions
3. **User Experience**: Remember your preferences and settings locally
4. **AI Processing**: If you provide an OpenAI API key, tab titles and domains are sent directly to OpenAI's servers (not to us)

## Data Storage and Security
- **100% Local Storage**: All data is stored locally on your device using Chrome's storage API
- **No External Servers**: We do not operate any servers or databases that store your data
- **No Data Transmission**: No data is sent to us or any third party (except OpenAI when you use AI features with your own API key)
- **Encryption**: Your OpenAI API key is encrypted using Chrome's secure storage
- **No Access**: We cannot access your browsing data, tabs, or any other information

## Third-Party Services

### OpenAI API (Optional)
- **When used**: Only when you provide your own OpenAI API key and use AI organization features
- **What's sent**: Tab titles and domain names (not full URLs) are sent directly from your browser to OpenAI
- **Your control**: You can use the extension without providing an API key (basic organization features available)
- **Billing**: You are billed directly by OpenAI for API usage. [View current pricing](https://openai.com/pricing)
- **Model used**: GPT-4 Turbo Preview
- **No other services**: We don't use any other third-party services

## Your Rights and Control
- **Complete Control**: You can clear all stored data at any time by uninstalling the extension
- **Feature Control**: You can disable specific features in the extension settings
- **API-Free Usage**: You can use the extension without providing an API key (with smart local organization)
- **Data Export**: You can export your workspace data from the extension settings
- **Instant Deletion**: Uninstalling the extension immediately removes all stored data

## What We DON'T Do
- We **never** collect, store, or have access to your browsing data
- We **never** transmit your data to our servers (we don't have any!)
- We **never** sell, trade, or share your data with anyone
- We **never** use your data for advertising, marketing, or analytics
- We **never** track your browsing behavior
- We **never** store your OpenAI API key on external servers

## Data Sharing
**We never share any data because we never have access to it.** All data processing happens locally on your device.

## Extension Permissions Explained
Our extension requests these permissions:
- **tabs**: To read tab information (titles, URLs) for organization
- **storage**: To save your workspaces and preferences locally
- **contextMenus**: To add right-click menu options
- **notifications**: To show organization completion notifications
- **alarms**: For auto-organization features
- **activeTab**: To interact with the current active tab
- **host_permissions**: To access tab information across all websites

## Changes to This Policy
We may update this privacy policy from time to time. We will notify you of any changes by updating the "Last updated" date and through the extension's update notifications.

## Contact
If you have questions about this privacy policy, please contact us at:
- **GitHub**: [https://github.com/mcsedition-corp/Context/issues](https://github.com/mcsedition-corp/Context/issues)
- **Email**: mcsedition@gmail.com

## Français

# Politique de Confidentialité pour Context - Gestionnaire d'Onglets Intelligent

*Dernière mise à jour : Janvier 2025*

## Vue d'ensemble
Context est une extension de navigateur qui vous aide à organiser intelligemment vos onglets grâce à l'IA. Nous nous engageons à protéger votre vie privée et à être transparents sur nos pratiques de données. **Toutes vos données restent sur votre appareil - nous ne stockons ni n'accédons jamais à vos informations.**

## Informations que nous collectons

### Informations sur les onglets
- **Ce que nous collectons** : URLs, titres et favicons de vos onglets ouverts
- **Pourquoi** : Pour analyser et organiser vos onglets en espaces de travail pertinents
- **Stockage** : **Stocké localement sur votre appareil uniquement** via l'API de stockage sécurisé de Chrome
- **Accès** : Seul vous avez accès à ces données

### Clé API (Optionnelle)
- **Ce que nous collectons** : Votre clé API OpenAI (si vous choisissez d'en fournir une)
- **Pourquoi** : Pour activer les fonctionnalités d'organisation par IA
- **Stockage** : **Chiffrée et stockée localement** dans le stockage sécurisé de Chrome sur votre appareil
- **Important** : Votre clé API n'est **jamais transmise à nous** - elle va directement de votre navigateur aux serveurs d'OpenAI
- **Coût** : Quand vous utilisez les fonctionnalités IA, vous payez OpenAI directement. [Voir les tarifs GPT-4 Turbo d'OpenAI](https://openai.com/pricing)

### Préférences utilisateur
- **Ce que nous collectons** : Vos paramètres (préférences d'organisation automatique, préférences de notification, thème, etc.)
- **Pourquoi** : Pour personnaliser votre expérience et mémoriser vos préférences
- **Stockage** : **Stocké localement sur votre appareil uniquement**

## Comment nous utilisons les informations
1. **Organisation des onglets** : Analyser le contenu des onglets localement pour créer des espaces de travail intelligents
2. **Persistance** : Sauvegarder localement les configurations d'espaces de travail pour les sessions futures
3. **Expérience utilisateur** : Mémoriser localement vos préférences et paramètres
4. **Traitement IA** : Si vous fournissez une clé API OpenAI, les titres et domaines des onglets sont envoyés directement aux serveurs d'OpenAI (pas à nous)

## Stockage et sécurité des données
- **Stockage 100% Local** : Toutes les données sont stockées localement sur votre appareil via l'API de stockage Chrome
- **Aucun Serveur Externe** : Nous n'exploitons aucun serveur ou base de données qui stocke vos données
- **Aucune Transmission** : Aucune donnée n'est envoyée à nous ou à des tiers (sauf à OpenAI quand vous utilisez les fonctionnalités IA avec votre propre clé API)
- **Chiffrement** : Votre clé API OpenAI est chiffrée via le stockage sécurisé de Chrome
- **Aucun Accès** : Nous ne pouvons pas accéder à vos données de navigation, onglets, ou autres informations

## Services tiers

### API OpenAI (Optionnelle)
- **Quand utilisée** : Uniquement quand vous fournissez votre propre clé API OpenAI et utilisez les fonctionnalités d'organisation IA
- **Ce qui est envoyé** : Les titres et noms de domaine des onglets (pas les URLs complètes) sont envoyés directement de votre navigateur à OpenAI
- **Votre contrôle** : Vous pouvez utiliser l'extension sans fournir de clé API (fonctionnalités d'organisation basique disponibles)
- **Facturation** : Vous êtes facturé directement par OpenAI pour l'utilisation de l'API. [Voir les tarifs actuels](https://openai.com/pricing)
- **Modèle utilisé** : GPT-4 Turbo Preview
- **Aucun autre service** : Nous n'utilisons aucun autre service tiers

## Vos droits et contrôle
- **Contrôle Complet** : Vous pouvez effacer toutes les données stockées à tout moment en désinstallant l'extension
- **Contrôle des Fonctionnalités** : Vous pouvez désactiver des fonctionnalités spécifiques dans les paramètres
- **Utilisation Sans API** : Vous pouvez utiliser l'extension sans fournir de clé API (avec organisation intelligente locale)
- **Export de Données** : Vous pouvez exporter vos données d'espaces de travail depuis les paramètres
- **Suppression Instantanée** : Désinstaller l'extension supprime immédiatement toutes les données stockées

## Ce que nous NE faisons PAS
- Nous ne **collectons jamais**, ne stockons jamais, ni n'avons accès à vos données de navigation
- Nous ne **transmettons jamais** vos données à nos serveurs (nous n'en avons pas !)
- Nous ne **vendons jamais**, n'échangeons jamais, ni ne partageons jamais vos données avec quiconque
- Nous n'**utilisons jamais** vos données pour la publicité, le marketing, ou l'analytique
- Nous ne **suivons jamais** votre comportement de navigation
- Nous ne **stockons jamais** votre clé API OpenAI sur des serveurs externes

## Partage de données
**Nous ne partageons jamais aucune donnée car nous n'y avons jamais accès.** Tout le traitement des données se fait localement sur votre appareil.

## Permissions de l'extension expliquées
Notre extension demande ces permissions :
- **tabs** : Pour lire les informations des onglets (titres, URLs) pour l'organisation
- **storage** : Pour sauvegarder vos espaces de travail et préférences localement
- **contextMenus** : Pour ajouter des options de menu clic-droit
- **notifications** : Pour afficher les notifications de fin d'organisation
- **alarms** : Pour les fonctionnalités d'organisation automatique
- **activeTab** : Pour interagir avec l'onglet actuellement actif
- **host_permissions** : Pour accéder aux informations des onglets sur tous les sites web
